Policy & Academic Use

Privacy Policy

This project stores and processes user data only for educational, testing, and project demonstration purposes.

Academic Notice

Deep Fake Shield is an academic platform. It is designed for educational demonstration, awareness, and basic forensic-style analysis. It is not intended to replace official forensic verification systems.

What Data Is Collected

  • User registration details such as username and email
  • Uploaded image, video, and audio files
  • Submitted text for analysis
  • Generated scan results and authenticity reports
  • User preferences like theme, language, and cursor settings

How Data Is Used

  • To perform media verification and scoring
  • To display dashboard scan history
  • To generate downloadable PDF reports
  • To improve project demonstration and analysis flow
  • To support academic review by teachers or evaluators

Storage Information

This platform uses a PostgreSQL and cloud storage.
