About DeepFake Shield

A practical media authenticity verification platform built using Django, computer vision, signal processing, and heuristic AI for engineering-level demonstration and deployment.

Project Objective

DeepFake Shield is designed to verify the authenticity of digital media including images, videos, audio, and text. It helps identify suspicious or AI-generated content using a hybrid approach combining metadata, visual artifacts, signal analysis, and backend intelligence.

The project is intended to be stable, educational, deployable, and presentation-ready for final-year engineering and board exam demonstrations.

System Modules

  • Image Authenticity Detection
  • Video Frame-Based Analysis
  • Audio Signal Authenticity Detection
  • Text AI Pattern Analysis
  • User Authentication & Scan History
  • PDF Report Generation
  • Admin Review Panel
